# NikhilOS

NikhilOS is a browser-based operating system inspired by modern desktop environments. It was built using HTML, CSS and JavaScript as part of the Hack Club WebOS mission.

## Features

* Draggable windows
* Terminal application
* Notes application with local storage
* Music player
* Calculator
* Settings panel
* Browser-style interface
* Glassmorphism inspired UI
* Live clock

## Technologies Used

* HTML
* CSS
* JavaScript
